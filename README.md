# 🚀 IntelliPath - AI-Powered Career Guidance System

## 📌 Project Description
IntelliPath is an AI-powered career guidance web application designed to help users discover the most suitable career paths based on their interests, learning style, and problem-solving approach. It provides personalized insights and structured career recommendations.

---

## 🎯 Key Features

- 🧠 **Learning & Interest Assessment**
  - Interactive questions to understand user preferences and thinking style

- 📊 **Career Path Recommendations**
  - Suggests roles like Frontend, Backend, and Full Stack Development

- 💡 **Personalized Insights**
  - Provides clarity, decision confidence, and optimized learning roadmap

- 🎨 **Modern UI/UX**
  - Clean, responsive, and user-friendly interface
